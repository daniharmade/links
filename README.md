# 🚀 Dani Harmade — Developer Links Page

A modern **developer links landing page** built using **HTML + TailwindCSS**.  
This page acts as a personal hub to share my portfolio, CV, research profiles, and social media in one place.

🔗 Live Demo  
https://daniharmade.vercel.app/links

---

## ✨ Features

- 🎨 Modern dark UI with TailwindCSS
- 🌌 Animated particle background
- 🧑 Developer profile card
- 🧰 Tech stack icons
- 🔗 Social media & professional links
- 📄 Portfolio & CV download
- 📧 Email call-to-action button
- 📱 Fully responsive layout
- 🖱 Smooth hover animations

---

## 🧰 Tech Stack

| Technology | Usage |
|------------|------|
| HTML5 | Page structure |
| TailwindCSS | Styling & layout |
| JavaScript | Particle animation |
| Particles.js | Background animation |
| Devicon | Tech stack icons |
| Font Awesome | Social media icons |

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/developer-links.git
```

Open the project folder:

```bash
cd developer-links
```

Run locally:

```
Open index.html in your browser
```
